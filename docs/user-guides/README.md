## Who Should Read These Guides

- **Sales Reps**: Daily users who manage opportunities, log activities, and track samples
- **Managers**: Team leads who review pipeline and generate reports
- **Admins**: System administrators who manage users and settings

## Need Help?

If you can't find what you're looking for in these guides:
1. Check the [FAQ](./faq.md) for common questions
2. Contact your system administrator
3. Review the [PRD](../PRD.md) for detailed feature specifications
