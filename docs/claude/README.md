## Key Business Terminology (from PRD)

- **Principal** = Food manufacturer (e.g., McCRUM, Rapid Rasoi)
- **Distributor** = Distribution company (e.g., Sysco, USF, GFS)
- **Customer/Operator** = Restaurant/foodservice end customer
- **Opportunity** = Deal linking Principal + Customer (+ optional Distributor)
