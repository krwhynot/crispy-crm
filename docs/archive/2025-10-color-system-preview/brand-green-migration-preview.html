8. Link Styling (WCAG Best Practice)

Underlined by default with modern CSS for accessibility

This is a paragraph with an inline link that demonstrates the new link styling. Notice the underline is offset from the text baseline for better readability. The underline thickens slightly on hover to provide clear feedback. Links use brand green color consistently throughout the application.

text-decoration-thickness: 1.5px

text-decoration-skip-ink: auto
Accessibility First: Per WCAG guidelines and industry research (even Figma designers acknowledge), underlines are necessary for accessible hyperlinks. We use modern CSS to make them elegant.
