Migration Recommendations

Action Required: The following hardcoded colors should be migrated to CSS variables:
  • Tag colors (10 hex values) → Create semantic tag variables
  • Note status colors (4 hex values) → Create status variables
  • Chart colors (5 hex values) → Use existing chart variables or create new ones
Unused Variables: The following CSS variables are defined but not actively used:
  • Chart colors (--chart-1 through --chart-5)
  • Sidebar colors (all --sidebar-* variables)
  • Popover colors (--popover, --popover-foreground)
Consider removing these to reduce complexity or implement them where hardcoded colors exist.
